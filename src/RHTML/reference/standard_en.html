Additional iconmenu :: General


This category contains frequently-used HTML elements that you may need on a regular basis. You can use their function by clicking on them. We tell you what each icon does and illustrate them as far as possible to make them understable. Globe also has an automatic help system about which you can read in more detail in the tutorial section.



Align left: By pushing this button Globe inserts the
<P ALIGN="left"></P> elements into the text at the position of the cursor and puts it between them. This way you can write the text aligned to left immediately. If you want to set additional paragraph properties put the cursor before the HTML starting element > and hit a space. Globe will help you to select parameters. You can read about this function in more detail in the tutorial section.
Align center: By pushing this button Globe inserts the
<P ALIGN="center"></P> elements into the text at the position of the cursor and puts it between them. This way you can write the text aligned to center immediately. If you want to set additional paragraph properties put the cursor before the HTML starting element > and hit a space. Globe will help you to select parameters. You can read about this function in more detail in the tutorial section.
Align right: By pushing this button Globe inserts the
<P ALIGN="right"></P> elements into the text at the position of the cursor and puts it between them. This way you can write the text aligned to right immediately. If you want to set additional paragraph properties put the cursor before the HTML starting element > and hit a space. Globe will help you to select parameters. You can read about this function in more detail in the tutorial section.
Align justify: By pushing this button Globe inserts the
<P ALIGN="justify"></P> elements into the text at the position of the cursor and puts it between them. This way you can write the text justified immediately. The justified text is supported by mozilla v0.9.9 under BeOS as well. If you want to set additional paragraph properties put the cursor before the HTML starting element > and hit a space. Globe will help you to select parameters. You can read about this function in more detail in the tutorial section.
Paragraph: By pushing this button Globe inserts the
<P></P> elements into the text at the position of the cursor and puts it between them. This way you can create a simple paragraph in your HTML document. If you want to set additional paragraph properties put the cursor before the HTML starting element > and hit a space. Globe will help you to select parameters. You can read about this function in more detail in the tutorial section.
Line break: By pushing this button Globe inserts the
<BR>element into the text. You can place a line break into your text where you want.
Space: By pushing this button Globe inserts the
&nbsp; special element into the text. You may need this when you want to place fixed number of spaces into the text. Large number of browsers show the exact number of spaces only in the case when this element is used.
Anchor: By pushing this button Globe inserts the
<A></A> elements into the text at the position of the cursor and puts it between them. If you want to set additional properties for it put the cursor before the HTML starting element > and hit a space. Globe will help you.
Image: By pushing this button Globe inserts the
<IMG> element into the text at the position of the cursor and puts it before the closing element. Then if you push a space Globe will help you to give additional properties.
Horizontal line: By pushing this button Globe inserts the
<HR> element into the text at the position of the cursor and puts it before the closing element. This way you can create a horizontal separating line in your HTML document. Then if you push a space Globe will help you to give additional properties.
DIV: By pushing this button Globe inserts the
<DIV></DIV> elements into the text at the position of the cursor and puts it between them. If you want to set additional properties concerning DIVISION element put the cursor before the HTML starting element > and hit a space. Globe will help you to select parameters. You can read about this function in more detail in the tutorial section.
