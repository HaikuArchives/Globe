Main toolbar


The main toolbar can be found at the upper part of the editing area next to the additional toolbars. By default it is on. By clicking on the different icons you can execute the functions which are in connection with file management and program controlling. The functions are the same as those of the menu bar except for the NetPositive icon. They operate the same way but you do not need to find them among the menus.



New:You can create a new document. It will be an empty and "noname" file.
Open: You can open and edit an existing file.
Save: You can save the edited document
Save all: You can save all the opened documents. If they were noname Globe will give you a chance to provide names to them one by one.
Close: You can close the document you are currently working on. If you have more than one file open the name of the current file appears in bold at the bottom of the window.
Quit:You can exit the program

If have not saved the changes you made the Globe warns you.
Cut: It cuts your selected text or code from the edited documentum. Then you can paste it by means of a paste statement or the proper keyboard shortcut.
Copy: It copies the selected text or code to the clipboard so the text will not be deleted from its original position. The copied text can then be pasted into any of edited documents. You should keep in mind that in case of each Cut and Copy statements the contents of the clipboard will be overwritten.

Paste:It puts the clipboard content at the position of the cursor.
Undo:You can cancel the last operation. If you click on it once more the cancelled operation will happen again.
NetPositive view: You can open the edited document in the NetPositive browser. Save the document before activating this function!
